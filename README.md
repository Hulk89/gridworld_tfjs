## usage

* open *.html with your browser. (chrome browser recommanded)

* game.html
  * You can play gridworld with this file.
* DeepSarsa.html
  * deep sarsa training. Training starts with loading. 
* DQN.html
  * deep q network training. Training starts after few iteration.
  * It cannot converge
